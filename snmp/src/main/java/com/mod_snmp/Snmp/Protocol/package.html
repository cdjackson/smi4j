The SNMP protocol stack. The package is the core of the SNMP protocol implementation and is used by the Snmp.Manager package. It is designed according the SNMP architecture framework as a multilingual SNMP protocol engine.

The protocol versions supported are:

  • SNMP version 1
  • SNMP version 2 community based (2C)
  • SNMP version 3, only the security level, privacy, is not implemented

    • The following security models are implemented:

  • Community, used in SNMP version 1, 2C and 3.
  • User based Security Model (USM) RFC XXXX

    • NOTE: The community based security model for SNMP version 3 may be either model 1 or 2. However, it is probably not supported by many SNMP agents and it is neither an security replacement for the SNMP versions 1 and 2C.
