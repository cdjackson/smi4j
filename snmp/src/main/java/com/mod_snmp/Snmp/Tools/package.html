The SNMP commad line applications. This package contains various command-line orienmted SNMP tools with which management information can be retrieved from a remote SNMP agent. The tools are build on top of the management application toolkit provided as the ' com.mod_snmp.Snmp.Manager' package and they can be used as examples to built more complex management applications. For all tools both the arguments and the managed instances/varbinds have an equal approach based on the need of the tool. For specific information read the descriptions of the classes.
